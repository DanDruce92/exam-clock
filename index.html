All groups start together — end times differ by allowance.
How to use:
  1. From Start — enter when the exam begins. Each group's end time is calculated forward.
  2. From End — enter the fixed finish time. Start times are calculated backward so everyone finishes together.
  3. Toggle Normal / 25% / 50% Extra Time to show only the groups in your room.
  4. Enable Change Pen to show a live reminder on each group when their extra time begins. Students with rest breaks get their own adjusted reminder.
  5. Use Add student for rest breaks to add individuals with timed breaks — end time and change pen time adjust automatically.
